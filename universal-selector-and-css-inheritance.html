Universal Selector and CSS Inheritance

I want all text have 60px large

We can use universal selector

span span in span
aside 
*.universal {
    font-size: 60px;
}

Or CSS inheritance

span span in span
aside 
.inheritance {
    font-size: 60px;
}

What is the difference?
